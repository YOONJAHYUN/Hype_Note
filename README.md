## 📖 프로젝트 개요

<a name="item-one"></a>

<div>

<strong>목표</strong>

- 문서 정리 + 웹 겁색 + GPT 서비스를 한 페이지에서 한번에 사용할 수 있도록 편의성 제공

- 트리 구조로 문서를 한 눈에 확인

- 그래프 구조를 통해 문서 간 유사도를 연결 + 공유 받은 문서와 내 문서를 연결

- 내가 정리한 문서를 AI가 퀴즈로 만들어줘 복습을 간편하게 할 수 있도록 편의성 제공
